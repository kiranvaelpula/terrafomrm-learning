## 🚀 Quick Commands for Your Journey

```bash
# Start your cluster
minikube start

# Create your first pod
kubectl run nginx --image=nginx

# Check what's running
kubectl get all

# View cluster info
kubectl cluster-info

# Get help
kubectl --help
```
