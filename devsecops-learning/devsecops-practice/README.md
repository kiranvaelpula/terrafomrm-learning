# DevSecOps Practice Labs

Hands-on exercises to build practical DevSecOps skills.

## 🎯 Overview

This directory contains practical labs, exercises, and projects to apply DevSecOps concepts.

## 📚 Lab Structure

### Lab 1: Basic Security Scanning
**Duration**: 30 minutes  
**Skills**: Trivy, SAST, Secrets scanning  
**Folder**: `lab-01-basic-scanning/`

### Lab 2: CI/CD Security Pipeline
**Duration**: 1 hour  
**Skills**: GitHub Actions, GitLab CI, Security gates  
**Folder**: `lab-02-cicd-security/`

### Lab 3: Container Security
**Duration**: 1 hour  
**Skills**: Docker security, Image scanning, Hardening  
**Folder**: `lab-03-container-security/`

### Lab 4: Kubernetes Security
**Duration**: 1.5 hours  
**Skills**: RBAC, Network Policies, Pod Security  
**Folder**: `lab-04-kubernetes-security/`

### Lab 5: Secrets Management
**Duration**: 1 hour  
**Skills**: Vault, AWS Secrets Manager, Encryption  
**Folder**: `lab-05-secrets-management/`

### Lab 6: IaC Security
**Duration**: 1 hour  
**Skills**: Terraform scanning, Policy as Code  
**Folder**: `lab-06-iac-security/`

### Lab 7: Security Monitoring
**Duration**: 1 hour  
**Skills**: Falco, Prometheus, Alerting  
**Folder**: `lab-07-security-monitoring/`

### Lab 8: Incident Response
**Duration**: 2 hours  
**Skills**: Detection, Analysis, Remediation  
**Folder**: `lab-08-incident-response/`

## 🚀 Quick Start

```bash
# Navigate to a lab
cd lab-01-basic-scanning

# Read instructions
cat README.md

# Run setup script
./setup.sh

# Complete exercises
# Check your work
./verify.sh
```

## 📋 Prerequisites

- Docker installed
- kubectl (for Kubernetes labs)
- Git
- Python 3.8+
- Basic Linux/Bash knowledge
