# DevOps Essentials Learning Path
## Master Git, Docker & Jenkins

A comprehensive, hands-on curriculum covering the three foundational tools of modern DevOps: Git for version control, Docker for containerization, and Jenkins for CI/CD automation.

---

## 🎯 What You'll Learn

This curriculum teaches you the essential DevOps tools that every developer and DevOps engineer needs to master:

- **Git**: Version control, collaboration workflows, branching strategies
- **Docker**: Containerization, image building, orchestration basics
- **Jenkins**: CI/CD pipelines, automation, deployment strategies

## 🗺️ Learning Path

### Beginner Path (Weeks 1-3)
**Goal:** Understand fundamentals and basic usage

1. **Week 1: Git Basics**
   - What is version control
   - Basic Git commands
   - First repository
   - Lab: Git fundamentals

2. **Week 2: Docker Basics**
   - What is containerization
   - Docker basics
   - First container
   - Lab: Build and run containers

3. **Week 3: Jenkins Basics**
   - What is CI/CD
   - Jenkins setup
   - First pipeline
   - Lab: Basic Jenkins job

### Intermediate Path (Weeks 4-7)
**Goal:** Learn workflows and best practices

4. **Week 4-5: Git Intermediate**
   - Branching strategies
   - Collaboration workflows
   - Advanced operations
   - Labs: Git workflows

5. **Week 6: Docker Intermediate**
   - Multi-stage builds
   - Docker Compose
   - Networking
   - Labs: Complex applications

6. **Week 7: Jenkins Intermediate**
   - Pipeline as code
   - Integration with Git/Docker
   - Testing automation
   - Labs: Complete pipelines

### Advanced Path (Weeks 8-10)
**Goal:** Master production patterns

7. **Week 8: Git Advanced**
   - Git internals
   - Performance optimization
   - Enterprise patterns
   - Lab: Complex workflows

8. **Week 9: Docker Advanced**
   - Optimization techniques
   - Security hardening
   - Orchestration intro
   - Lab: Production containers

9. **Week 10: Jenkins Advanced**
   - Distributed builds
   - Pipeline libraries
   - Monitoring
   - Lab: Enterprise pipeline

10. **Integration Projects**
    - Complete CI/CD pipeline
    - Git → Docker → Jenkins integration
    - Real-world scenarios
